Blogging on the Phone

Had finally some time to figure out a good process for blogging on the phone. I’m using Termux to keep track of the files. At first, I used the integrated vim. This is not that nice because the Touchscreen is not well working here with vim.

In between, I’ve migrated my hackmd to Hedge Doc. However, it still has the same problems as hackmd. Hence, using it for blogging wasn’t an option.

For a while, things remained on hold, until I realized that I need to use a more proper editor, vim will do. Once the CI/CD will be deployed, I can push blog posts from the phone and get them deployed!
